# ShortcutManager (快速启动器)

ShortcutManager 是一款基于 WPF (.NET 8.0) 开发的轻量级 Windows 桌面快速启动工具。它旨在帮助用户高效地组织和快速访问常用的应用程序、文件、文件夹以及网站链接，提供类似于 macOS Launchpad 或各类 Dock 工具的流畅体验。

## ✨ 功能特点 (Features)

*   **拖拽管理 (Drag & Drop)**
    *   直接将 `.exe`、文件、文件夹或快捷方式拖入窗口即可添加。
    *   支持拖拽排序：无论是左侧分类还是右侧的图标，都可以通过拖拽自由调整顺序。
*   **多类型支持**
    *   支持应用程序 (`.exe`)、脚本 (`.bat`, `.cmd`, `.ps1`)。
    *   支持各类文档 (`.txt`, `.doc`, `.pdf` 等)。
    *   支持文件夹快捷入口。
    *   支持网址链接 (`.url`)。
*   **个性化与主题 (Theming)**
    *   **明亮/暗黑模式**：内置 Light 和 Dark 两套基础外观，可跟随系统自动切换。
    *   **多彩主题**：提供 蓝色、绿色、橙色、紫色、粉色、红色、黄色 等多种强调色。
    *   **自定义**：右键菜单可快速切换主题风格。
*   **高效交互**
    *   **全局热键**：默认使用 `Alt + Space` 快速呼出/隐藏窗口（支持自定义热键）。
    *   **自动隐藏**：窗口失去焦点自动最小化到托盘（可选）。
    *   **流畅动画**：使用 `FluidWrapPanel` 实现图标流式布局与平滑重排动画。
*   **分类管理**
    *   支持创建多个侧边栏分类，将工作、娱乐、工具等不同用途的软件分组管理。

## 🛠️ 安装与构建 (Build & Install) 

### 环境要求
*   Windows 10 或 Windows 11
*   [.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

### 构建步骤
1.  克隆本项目到本地：
    ```bash
    git clone https://github.com/your-repo/ShortcutManager.git
    ```
2.  使用 Visual Studio 2022 打开 `ShortcutManager.sln`。
3.  选择 `Release` 模式并生成解决方案。
4.  运行生成的 `ShortcutManager.exe`。

或者使用 .NET CLI：
```bash
cd ShortcutManager
dotnet build -c Release
```

## 📖 使用指南 (Usage)

### 添加启动项
*   **方法 1**：直接将桌面或资源管理器中的文件/文件夹拖拽到右侧图标区域。
*   **方法 2**：在右侧空白区域 **右键单击**，选择“添加快捷方式”或“添加文件夹”。

### 管理分类
*   **新增分类**：点击左下角的“新增分类”菜单项，或在左侧列表空白处右键。
*   **管理分类**：在左侧分类名称上 **右键单击**，可进行“重命名”或“删除”操作。

### 更多设置
在右侧图标区域的 **空白处右键单击**，可以调出主菜单：
*   **添加/管理内容**：添加快捷方式、文件夹、命令。
*   **主题切换**：选择明亮/暗黑模式及主题色。
*   **热键设置**：开启/关闭热键，或设置新的全局快捷键。

## ⚙️ 数据存储
*   配置文件默认保存在程序同级目录下的 `config.json`（快捷方式数据）和 `settings.json`（窗口配置）。
*   程序为绿色免安装版，方便随身携带。
