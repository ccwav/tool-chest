# tool-chest

存放个人开发的各类实用小工具。

## 工具列表

### [ShortcutManager](ShortcutManager/README.md) — 快速启动器

基于 WPF (.NET 8.0) 开发的轻量级 Windows 桌面快速启动工具。  
支持拖拽管理、多类型文件/链接、全局热键、明暗主题切换与分类管理。

### [Gemini_Pipeline](Gemini_Pipeline/README.md) — Gemini 管道

用于 Open WebUI 的 Google Gemini Pipeline。  
支持多 API Key 轮询、流式响应、Thinking 模式、图片压缩与重试策略等高级特性。
